# AiBook

AI를 다시 학습하기 위한 개인 커리큘럼 북입니다.

대학에서 배운 개론과 기초, 프로젝트 경험을 다시 꺼내기 위해 필요한 내용을 커리큘럼 순서로 정리합니다. 목표는 흩어진 자료를 단순히 모으는 것이 아니라, 기초 복구부터 생성형 AI 실전까지 이어지는 학습 경로를 책처럼 읽을 수 있게 만드는 것입니다.

## 책의 방향

- 잊어버린 개념을 다시 떠올릴 수 있게 핵심 정의와 직관을 먼저 정리합니다.
- 수학, Python, 데이터 처리, 머신러닝, 딥러닝, LLM을 순서대로 연결합니다.
- 각 장은 "왜 필요한가", "무엇을 알아야 하는가", "어떻게 확인할 것인가"를 중심으로 작성합니다.
- 외부 자료를 참고한 경우 출처를 명확히 남깁니다.

## 전체 흐름

1. 기초 복구: 학습 환경, Python, 수학, 데이터 다루기
2. 머신러닝: 지도학습, 비지도학습, 평가, 전통적 모델
3. 딥러닝: 신경망, 역전파, 주요 모델 구조
4. LLM과 생성형 AI: Transformer, 사전학습, RAG, Agent
5. 프로젝트: 배운 내용을 작은 산출물로 검증

## 작성 규칙

- 원고는 `docs/` 아래 Markdown 파일로 작성합니다.
- 목차는 `mkdocs.yml`의 `nav`에서 관리합니다.
- 직접 인용, 요약, 재서술한 외부 자료는 문서 하단의 `출처와 참고 자료`에 기록합니다.
