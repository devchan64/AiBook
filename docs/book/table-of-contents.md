## 전체 흐름

```mermaid
flowchart LR
  A[AI 개론과 지형도]
  B[기초 복구]
  C[머신러닝]
  D[딥러닝]
  E[LLM과 생성형 AI]
  F[프로젝트]

  A --> B --> C --> D --> E --> F
```
