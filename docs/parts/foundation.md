# Part 1. 기초 복구

AI를 다시 공부하기 전에 학습 환경, Python, 데이터 처리, 수학 기초를 먼저 정리합니다. 이 파트의 목적은 모든 내용을 깊게 파는 것이 아니라, 머신러닝과 딥러닝을 읽고 구현할 수 있을 정도의 기반을 회복하는 것입니다.

## 다룰 내용

- Python 문법과 개발 환경
- NumPy, Pandas, Matplotlib
- 선형대수의 핵심 개념
- 미분과 경사하강법
- 확률과 통계의 기본 개념
- 데이터셋을 읽고 나누고 시각화하는 방법

## 완료 기준

- 간단한 데이터셋을 불러와 탐색할 수 있다.
- 벡터와 행렬 연산이 모델 계산에서 어디에 쓰이는지 설명할 수 있다.
- 손실 함수와 경사하강법의 역할을 설명할 수 있다.
