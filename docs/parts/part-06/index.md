# Part 6. 프로젝트

프로젝트 파트에서는 앞에서 배운 내용을 작은 산출물로 검증합니다. 각 프로젝트는 개념 복습, 구현, 결과 해석, 개선 방향을 포함하도록 정리합니다.

## 프로젝트 후보

- 데이터 분석 미니 프로젝트
- 회귀 또는 분류 모델 만들기
- 이미지 분류 모델 만들기
- 텍스트 분류 모델 만들기
- 간단한 RAG 챗봇 만들기
- 문서 기반 Q&A 시스템 만들기
- 도구를 사용하는 AI Agent 만들기

## 프로젝트 문서 형식

각 프로젝트는 다음 형식으로 작성합니다.

- 문제 정의
- 데이터 또는 입력
- 접근 방법
- 구현
- 결과
- 한계와 개선점
- 출처와 참고 자료

## 출처와 참고 자료

이 문서는 현재 프로젝트 후보를 정리한 자체 초안이며, 외부 자료를 직접 인용하지 않았습니다.
