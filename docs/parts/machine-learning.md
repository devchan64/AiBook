# Part 3. 머신러닝

머신러닝 파트에서는 데이터로부터 규칙을 학습한다는 것이 무엇인지 정리합니다. 전통적인 모델을 먼저 다루는 이유는 딥러닝 이전에 학습, 일반화, 평가, 과적합 같은 기본 개념을 분명하게 이해하기 위해서입니다.

## 다룰 내용

- 지도학습, 비지도학습, 강화학습의 차이
- 학습 데이터, 검증 데이터, 테스트 데이터
- 회귀와 분류
- 선형회귀와 로지스틱 회귀
- 결정트리, 랜덤포레스트, 그래디언트 부스팅
- 클러스터링과 차원 축소
- 평가 지표와 교차 검증

## 완료 기준

- 문제 유형에 맞는 모델과 평가 지표를 선택할 수 있다.
- 과적합과 일반화의 차이를 설명할 수 있다.
- scikit-learn으로 기본 학습 파이프라인을 구성할 수 있다.
