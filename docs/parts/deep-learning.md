# Part 3. 딥러닝

딥러닝 파트에서는 신경망이 데이터를 어떻게 표현하고 학습하는지 정리합니다. 단순한 퍼셉트론에서 시작해 역전파, 최적화, 주요 모델 구조로 이어갑니다.

## 다룰 내용

- 퍼셉트론과 다층 신경망
- 활성화 함수와 손실 함수
- 역전파
- 옵티마이저
- 정규화와 드롭아웃
- CNN
- RNN, LSTM, GRU
- Attention과 Transformer의 연결 지점

## 완료 기준

- 신경망의 순전파와 역전파 흐름을 설명할 수 있다.
- 손실 함수, 옵티마이저, 학습률의 역할을 구분할 수 있다.
- PyTorch로 간단한 모델을 학습시킬 수 있다.
