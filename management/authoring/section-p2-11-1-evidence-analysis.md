## 본문에 반영한 안전한 설명

- NumPy는 과학과 공학에서 널리 쓰이는 오픈소스 Python 라이브러리이며, 숫자 배열 계산을 위한 핵심 도구로 소개할 수 있다.
- NumPy 배열은 같은 종류의 숫자를 정해진 모양으로 놓고 계산하기 위한 구조로 설명할 수 있다.
- Python 리스트는 범용 컨테이너이고, NumPy 배열은 숫자 계산에 특화된 배열로 설명할 수 있다.
- AI 실습에서 NumPy를 먼저 보는 이유는 shape, 벡터, 행렬, 위치별 연산, 행렬 곱 같은 계산 감각을 작은 예제로 확인하기 위해서라고 설명했다.
- 같은 `+` 기호라도 Python 리스트에서는 연결, NumPy 배열에서는 위치별 덧셈으로 읽힐 수 있음을 도식과 예제로 설명했다.
- `features @ weights` 예제는 `(3, 2) @ (2,) -> (3,)`의 shape 흐름으로 설명했다.
- 1차원 배열은 벡터로, 2차원 배열은 행렬로 읽을 수 있다.
- `shape`은 배열의 모양을 알려 주며, 계산 가능 여부를 판단하는 기본 정보다.
- `ndim`은 차원 수, `dtype`은 배열 원소의 데이터 타입을 알려 준다.
- AI 실습에서 입력, 특징, 가중치, 출력은 NumPy 배열의 모양으로 자주 나타난다.
