# P2-6.3 근거 분석: 경사하강법의 직관

## Section 목적

P2-6.3은 손실 함수와 목적 함수가 정해진 뒤, 모델 파라미터를 어떻게 반복적으로 조정하는지 설명하는 절이다.

중심 질문:

```text
경사하강법은 목적 함수를 줄이기 위해 파라미터를 어떤 방향으로 얼마나 움직이는가?
```

편집 흐름:

```text
손실을 줄이고 싶은 문제
-> 그래디언트 반대 방향
-> 학습률
-> 반복 업데이트
-> 한계와 주의점
```

## 확인한 근거

### Ian Goodfellow, Yoshua Bengio, Aaron Courville, Deep Learning, Chapter 8

- URL: https://www.deeplearningbook.org/contents/optimization.html
- 로컬 파일: `.tmp/section-p2-6-1-evidence/deeplearningbook-optimization.html`
- 확인 날짜: 2026-06-24

검토 내용:

- 딥러닝 학습에서 신경망 파라미터를 조정해 비용 함수(cost function)를 줄이는 최적화 문제가 중요하다는 설명을 확인했다.
- 머신러닝 학습의 최적화가 순수 최적화와 다르게, 훈련 기준을 줄여 일반화 성능이 좋아지기를 기대하는 간접 문제라는 점을 확인했다.
- 본문에는 고급 최적화 알고리즘, 확률적 경사하강법, 적응형 학습률 방법은 반영하지 않고, 그래디언트 기반 반복 업데이트의 입문 직관만 반영했다.

### Stephen Boyd, Lieven Vandenberghe, Convex Optimization

- URL: https://web.stanford.edu/~boyd/cvxbook/bv_cvxbook.pdf
- 로컬 파일: `.tmp/section-p2-6-1-evidence/boyd-vandenberghe-convex-optimization.pdf`
- 확인 날짜: 2026-06-24

검토 내용:

- 최적화 문제가 목적 함수와 변수, 제약의 관점에서 정리된다는 점을 P2-6.1에 이어 확인했다.
- 본문에는 볼록성, 수렴 보장, 세부 알고리즘 증명을 반영하지 않았다.

## P2-4.5와의 경계

P2-4.5는 그래디언트가 왜 생소할 수 있는지, 고등학교 미분 기억에서 다변수 미분으로 어떻게 넘어가는지 설명하는 보충수업이다.

P2-6.3은 같은 내용을 반복하지 않고 다음 역할에 집중한다.

```text
손실 함수와 목적 함수가 생겼다.
-> 그 기준을 줄이기 위해 파라미터를 반복적으로 조정한다.
-> 이 반복 방법의 대표적 직관이 경사하강법이다.
```

따라서 P2-6.3에서는 편미분 계산과 역전파 계산 절차를 다루지 않았다.

## 표현 정리

본문에 사용한 안전한 표현:

```text
경사하강법은 손실을 낮추기 위해 파라미터를 조금씩 바꾸는 반복 방법이다.
```

```text
그래디언트는 현재 위치의 방향 정보이며, 손실을 줄이려면 보통 그래디언트 반대 방향으로 이동한다.
```

```text
학습률은 한 번에 얼마나 움직일지 정하는 값이다.
```

피한 표현:

```text
경사하강법은 항상 전역 최적해를 찾는다.
그래디언트만 알면 학습은 자동으로 성공한다.
학습률은 크게 잡을수록 항상 좋다.
역전파와 경사하강법은 같은 말이다.
```
