# P3-8.1 모델 선택(model selection) - 근거 검토 메모

## 검토 목적

- 모델 선택을 초심자 기준에서 `문제와 제약에 맞는 후보군을 좁히는 일`로 설명한다.
- 모델 선택이 단지 성능표 비교가 아니라, 문제 유형, 데이터 조건, 설명 가능성, 운영 조건을 함께 보는 일이라는 점을 정리한다.
- 뒤 절의 기준 모델, 튜닝, 알고리즘 입문과 자연스럽게 연결한다.

## 핵심 근거

1. Jie Ding, Vahid Tarokh, Yuhong Yang, `Model Selection Techniques -- An Overview`
   - URL: https://arxiv.org/abs/1810.09583
   - 확인 날짜: 2026-06-26
   - 반영 포인트:
     - 모델 선택은 통계와 머신러닝 전반에서 핵심 단계다.
     - 하나의 기준이 아니라 다양한 목적과 철학이 존재한다.
   - 주의:
     - 본문에서는 정보 기준, 큰 표본 성질, 세부 이론은 다루지 않는다.

2. Sebastian Raschka, `Model Evaluation, Model Selection, and Algorithm Selection in Machine Learning`
   - URL: https://arxiv.org/abs/1811.12808
   - 확인 날짜: 2026-06-26
   - 반영 포인트:
     - 평가, 모델 선택, 알고리즘 선택은 구분되는 문제다.
     - 올바른 실험 설계와 비교 절차가 중요하다.
   - 주의:
     - 이 절에서는 검정 절차보다 `후보군을 세우는 사고`를 우선 설명한다.

## 본문 일반화 원칙

- 모델 선택을 `정답 모델 찾기`보다 `후보군 좁히기`로 설명한다.
- 문제 유형, 데이터 조건, 설명 가능성, 운영 제약을 함께 묻는 구조로 정리한다.
- 각 알고리즘의 상세 수학은 뒤 장으로 넘기고, 이 절은 큰 지도를 주는 역할에 머문다.
- 학술적으로는 모델 선택을 `후보 모형 가운데 목적과 제약에 맞는 모형을 고르는 단계`로 설명하되, 초심자 본문에서는 `실험해 볼 후보군을 세우는 일`로 일반화한다.
- 커리큘럼적으로는 P3-4~P3-7의 결과를 모아 P3-8.2와 P3-9, P3-10 이후로 넘기는 전환 절이라는 점을 드러낸다.

## 도메인 경계

- 이 절은 선택 기준의 큰 틀을 다룬다.
- baseline 구성은 P3-8.2에서 별도로 다룬다.
- 하이퍼파라미터 조정은 P3-9로 넘긴다.
- 개별 알고리즘의 내부 원리와 장단점은 P3-10 이후 절에서 구체화한다.
